# Question 5 Explanation

- Command: lsblk  
  This listed the available storage devices and their partitions so the storage layout could be assessed.
- Command: mount | head  
  This showed the mounted filesystems and helped identify where storage was attached in the environment.
- Command: df -h and df -i  
  These commands reported disk usage and inode utilization, which are key indicators of storage health.
- Command: report summary  
  The observations and recommendations summarize the storage state and give practical next steps for efficient use.
