# Question 1 Explanation

- Command: whoami  
  This verified the active Linux username and confirmed the account currently in use.
- Command: groups  
  This listed the groups associated with the user account and showed the account’s permissions context.
- Command: echo "$SHELL"  
  This displayed the default shell environment for the current session.
- Command: pwd  
  This confirmed the working directory so the session location was known.
- Command: ls -ls  
  This showed the files and directories present in the workspace and their current metadata.
- Command: curl ...  
  This tested outbound network access to YouTube and verified that the environment could reach the internet.
